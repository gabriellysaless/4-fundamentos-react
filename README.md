# ⚛️ Fundamentos de React

> **Repositório destinado aos estudos e exercícios desenvolvidos durante o módulo de Fundamentos de React da EBAC.**

Este repositório reúne os projetos e exercícios realizados durante o módulo, abordando os principais conceitos do React, componentização, gerenciamento de estado e desenvolvimento de interfaces modernas.

O objetivo é documentar minha evolução no aprendizado, consolidar os fundamentos da biblioteca e servir como material de consulta durante minha formação em Desenvolvimento Front-end.

## ⚙️ Como executar

Cada projeto possui sua própria estrutura.

Entre na pasta do projeto desejado:

```bash
cd nome-do-projeto
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
